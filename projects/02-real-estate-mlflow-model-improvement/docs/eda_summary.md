# EDA Summary

- The target price distribution is right-skewed and contains low-value outliers.
- Missing values were not present in the inspected dataset.
- Area and room-count features had the strongest observed relationship with price.
- Constant features such as `has_elevator`, `is_apartment`, and `studio` were candidates for removal.
- Candidate engineered features included building age, relative floor, first/last-floor flags, and area interactions.
